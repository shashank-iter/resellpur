
# Resellpur Landing Page
- Landing page for resellpur assginment.

## Tech Stack

- React
- Tailwind CSS
- Vite

## How to use

1. Clone the repo

``` bash
git clone [this repo]
```

2. Install dependencies

``` bash
npm install
```

3. Start the dev server

``` bash
npm run dev
```
